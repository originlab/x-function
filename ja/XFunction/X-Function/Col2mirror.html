メニュー情報

ワークシート：ミラー列に変換

概要

参照列をミラー列に変換します。

コマンドラインでの使用法

1. col2mirror scv:=0 colcpy:=1;

2. col2mirror colcpy:=1 label:=1 -u;

3. col2mirror scope:=1;

Xファンクションの実行オプション

スクリプトからXファンクションにアクセスする場合、追加のオプションスイッチについてのページを参照してください。

列の数式 scv

入力

int

1
 他の列を参照する数式（列値の設定またはF(x)）を使用して入力された列を変換するかどうかを指定します。

使用できる値： 0=false, 1=true.

列コピー出力 colcpy

入力

int

0
 ワークシート：列のコピーによって作成された列を変換するかどうかを指定します。

使用できる値： 0=false, 1=true.

変換範囲 scope

入力

int

2
 検索と変換の範囲を指定します。以下のオプションを利用できます。
  • sh:アクティブシート {0}
  • bk:アクティブブック {1}
  • fd:アクティブフォルダ {2}
  • sf:アクティブフォルダ(再帰的に適用) {3}
  • proj:プロジェクト{4}
ソース列ラベルを使用 label

入力

int

1
 列の数式が選択されているとき、このチェックボックスをオンにすると元の列からラベル行がコピーされます。列コピー出力において、元の列のラベルを使用するかどうかは、列のコピーダイアログ内のラベルのコピーブランチでどのラベル行が選択されているかによって制御されます。

使用できる値： 0=false, 1=true.

変換された列の一覧を出力 dump

入力

int

0
 変換された列の一覧を出力するかどうかを指定します。以下のオプションを利用できます。
  • none:なし {0}
  • msg:メッセージログに出力 {1}
  • note:ノートウィンドウに出力 {2}
  • pad:メモ帳に出力 {3}

説明

Origin 2026bで、元の列を参照するミラー列が導入されました。以前のバージョンで作成されたリンク列（列の数式（単純な参照で、計算が含まれないもの）または列のコピーダイアログによるもの）については、このXファンクション "col2mirror" を使用することで、指定した範囲をスキャンし、これらのリンク列をミラー列に変換することができます。
