メニュー情報

ファイル：新規作成：ワークブック：新規データテーブル

概要

テーブルの種類やその他の設定を選択して、新しいワークシートまたはデータテーブルを作成します。

追加の情報

必要なOriginのバージョン: 2026b以降

コマンドラインでの使用法

1. newtable type:=free xy:=XY* cols:=10;

2. newtable type:=x names:="Event;Group" xt:=1 xname:=ID;

3. newtable type:=one names:="Event;Group" xt:=1;

4. newtable names:="Event;Group" xname:=ID;

5. newtable type:=con;

6. newtable type:=suv names:="Event;Group" xt:=1 etime:=Months;

テーブルの種類 type

入力

int

3
 新しく作成するシートのテーブルの種類を指定します。オプションリスト:
  • free:自由形式(ワークシート) {0}
  • x:XY {1}
  • one:列 (1方向) {2}
  • two:グループ化(2方向) {3}
  • con:分割表 {4}
  • suv:生存表 {5}

テーブルの種類の詳細はこのページを参照してください。

列属性の設定 xy

入力

string

XY
 すべてのデータ列の列属性を指定します。例えば、XYを入力すると、列のプロット属性は"XYXYXY"の繰り返しパターンに設定されます。この文字列では、各文字が1つの列の属性に対応しています。以下をご覧ください。

'X' - X
'Y' - Y
'Z' - Z
'E' - Yエラー
'L' - ラベル
'M' - Xエラー
'N' - 無属性
