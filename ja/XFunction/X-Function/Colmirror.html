メニュー情報

ワークシート：類似シートからミラー列を作成

概要

類似シートから列を収集してミラー列を作成します。

コマンドラインでの使用法

1. colmirror irng:=[Book1]Sheet1!Col(1) orng:=[Book1]Sheet1!Col(2);

2. colmirror irng:=col(A) orng:=col(B) scope:=2 match:=1 unit:=1;

3. colmirror irng:=[Book1]Sheet1!Col(2) orng:=[Book2]Sheet1!Col(2) xcol:=1 bkrow:=1

スクリプトからXファンクションにアクセスする場合、追加のオプションスイッチについてのページを参照してください。

入力

int

0
 ソース列がY列の場合、このオプションを使用して、対応するX列も一緒にコピーするかどうかを制御できます。使用できる値：
  • no:なし {0}

  • this:共通X列 {1}
ソース列に対応するX列をコピーし、全てのミラー列の共通X列として使用します。出力はXYYY...という形になります。

各ミラー列に対して、自身のX列をコピーします。出力はXYXYXY...という形になります。

説明

Origin 2026bで、元の列を参照するミラー列が導入されました。このXファンクション関数 "colmirror" は、列ラベル行や列属性など、特定の条件に一致する複数のシートからミラー列を一括作成し、それらを1つのワークシートにまとめるために使用します。
