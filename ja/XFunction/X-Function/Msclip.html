メニュー情報

行列: 欠損をクリップ

概要

シート内の欠損値のある行列を他の行列でクリップ

コマンドラインでの使用法

msclip ims:=[Test1]Msheet1 imo:=[MBook2]pre!1 oms:=[<new>];

Xファンクション実行オプション

スクリプトからXファンクションにアクセスする場合、追加のオプションスイッチについてのページを参照してください。

変数

表示
名 		変数

と

クリップする行列 ims

入力

MatrixLayer

<active>

クリップする行列を指定します。

欠損値の行列 imo

入力

MatrixObject

<unassigned>

欠損値の行列を指定します。この行列の欠損値に応じてクリップする行列(ims)をクリップします。

ピクセル pixel

入力

int

0

ピクセルの定義方法を決定する設定します。

  • 0 = POINT:ポイント
ピクセルを単一の点として扱います。
  • 1 = AREA:領域
ピクセルを長方形の領域として扱います。
出力行列 oms

出力

MatrixLayer

[<new>]

出力する行列を指定します。

シンタックスはこちらを参照してください。

説明

このXファンクションを使って、別の行列の欠損値に基づいて行列をクリップすることができます。
